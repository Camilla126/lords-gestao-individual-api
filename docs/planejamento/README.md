# Planejamento por etapa

Cada **etapa** vale uma pasta em **`etapas/`** com só um **`plano.md`** (leve — sem duplicar tudo dos planeamentos grandes). Convenções em [`etapas/README.md`](etapas/README.md) e modelo em [`etapas/template-plano.md`](etapas/template-plano.md).

Os documentos **gerais** do produto/API continuam na **raiz** do repositório (`lords-mobile-*-planejamento.md`). O contrato das rotas implementadas documenta-se em [`api-rotas.md`](api-rotas.md).
